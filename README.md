# Flask To-Do App

This is a simple To-Do web application built with Flask. It allows users to add, update, and delete tasks easily, helping them manage daily activities in a clean and minimal interface.

The backend uses Flask and Flask-SQLAlchemy with a SQLite database to store tasks. Each task includes content and timestamps, making it easy to track and manage.

The frontend is built with Jinja2 templates and styled using Bootstrap, providing a responsive and user-friendly design. Flash messages are used to give feedback after actions like creating or deleting tasks.

This project is a great starting point for beginners learning Flask and web development.
Learn more: Flask ([https://flask.palletsprojects.com/](https://flask.palletsprojects.com/)), SQLAlchemy ([https://www.sqlalchemy.org/](https://www.sqlalchemy.org/)), Bootstrap ([https://getbootstrap.com/](https://getbootstrap.com/)) 🚀
